## Frontend Ekibi Ne Yapacak?

Frontend ekibinin işi:

```text
1. Kullanıcıdan fotoğraf alma.
2. /api/upload ile backend’e gönderme.
3. /api/process için eski ve yeni param formatını destekleme.
4. Efekt sliderları oluşturma.
5. Renk paletleri oluşturma.
6. Asset seçimi UI’ı yapma.
7. Before/after preview yapma.
8. Hata mesajlarını düzgün göstermek.
```

Frontend’in göndereceği legacy format şu anda korunmalı:

```json
{
  "mode": "expression",
  "params": {
    "smile_intensity": 0,
    "eyebrow_height": 0,
    "lip_intensity": 0,
    "face_slimming": 0,
    "aging_intensity": 1,
    "item_type": "glasses"
  }
}
```

Yeni effect engine için önerilen format:

```json
{
  "effects": {
    "skin_smooth": {
      "enabled": true,
      "strength": 0.5
    },
    "lipstick": {
      "enabled": true,
      "color": "#A02040",
      "intensity": 0.7
    },
    "hair_color": {
      "enabled": true,
      "color": "#7B3FE4",
      "intensity": 0.75
    },
    "accessories": {
      "enabled": true,
      "items": [
        {
          "type": "glasses",
          "asset_path": "assets/glasses/example.png",
          "scale": 1.0
        },
        {
          "type": "left_earring",
          "asset_path": "assets/earrings/example.png",
          "scale": 0.12,
          "offset_y_ratio": 0.07
        },
        {
          "type": "right_earring",
          "asset_path": "assets/earrings/example.png",
          "scale": 0.12,
          "offset_y_ratio": 0.07
        },
        {
          "type": "necklace",
          "asset_path": "assets/necklaces/example.png",
          "scale": 1.1,
          "offset_y_ratio": 0.14
        }
      ]
    }
  }
}
```

Frontend ekibi şunu bilmeli:

```text
- Accessory assetleri PNG + alpha channel olmalı.
- Ruj/saç/makyaj renkleri hex color olarak gönderilebilir.
- Slider değerleri 0.0 - 1.0 aralığında tutulmalı.
- Eski mode sistemi kırılmamalı.
```

Profesyonel FaceApp seviyesinde expression için sadece landmark warp yetmez. Generative model gerekir. Bu yüzden LivePortrait mantıklı ama bridge işi ayrı geliştirme ister.

# FaceApp Seviyesine Göre Eksikler

Dürüst durum:

```text
Bu proje şu an FaceApp seviyesinde değil.
```

FaceApp seviyesinde bir sistem için gereken ama bizde henüz tam olmayan şeyler:

## 1. Gerçek generative face editing

FaceApp tarzı efektler genelde sadece landmark warp değildir.

Gerekenler:

```text
- identity preserving generative model
- aging/de-aging model
- expression model
- makeup transfer/refinement model
- face restoration model
- artifact removal
```

Bizde durum:

```text
- slotlar hazır
- weightler hazır
- bazı temel effectler hazır
- full inference bridge çoğunda yok
```

## 2. Seamless blending

Profesyonel sonuç için generated yüzü orijinal fotoğrafa doğal yapıştırmak gerekir.

Eksik parçalar:

```text
- face crop align
- inverse transform
- Poisson blending veya advanced alpha blending
- skin color correction
- lighting match
- background preservation
```

## 3. Occlusion handling

Aksesuar/makyajda önemli:

```text
- saç önde mi arkada mı?
- küpe saçın arkasında mı?
- kolye çenenin altında mı?
- gözlük camı göz üstüne nasıl oturuyor?
```

Bizde:

```text
- anchor sistemi var
- maskeler var
- gerçek depth ordering henüz yok
```

## 4. Multi-pose robustness

FaceApp tarzı uygulamalar farklı pozlarda çalışmalı:

```text
- önden yüz
- hafif yan yüz
- düşük ışık
- gözlük/saç kapatma
- düşük kalite fotoğraf
```

Bizde:

```text
- MediaPipe stabil
- DECA var
- ama pose-specific asset fitting ve robust fallback eksik
```

## 5. Model optimization

Profesyonel app için hız gerekir.

Eksikler:

```text
- GPU inference optimization
- model caching
- lazy loading
- async queue
- batch/test pipeline
- memory limit yönetimi
```

Şu an CPU’da bazı şeyler çalışır ama SAM/LivePortrait gibi modeller ağır olabilir.

## 6. Frontend UX

FaceApp seviyesinde his için frontend çok önemli:

```text
- hızlı preview
- loading state
- effect intensity slider
- undo/redo
- before/after compare
- preset thumbnails
- asset marketplace/selector
```

Backend core olsa bile UX zayıfsa ürün kötü görünür.
