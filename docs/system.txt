## Important Design Decision

DECA, MediaPipe’ın yerine geçmez.

Doğru kullanım:

```text
MediaPipe
  -> 2D anchor
  -> gözlük/küpe/kolye/toka/piercing yerleşimi
  -> hızlı landmark
  -> frontend preview / realtime temel

BiSeNet face parsing
  -> saç maskesi
  -> cilt maskesi
  -> dudak maskesi
  -> kulak maskesi
  -> boyun maskesi
  -> göz / kaş maskesi

DECA / FLAME
  -> gerçek 3D mesh
  -> 5023 vertices
  -> 9976 faces
  -> future geometry / expression / reshape foundation
````

Yani gözle görülen aksesuar yerleşiminde MediaPipe anchorları kullanılmaya devam eder. DECA ise gerçek 3D yüz geometrisi için sisteme eklenmiştir.

## MediaPipe Face Detection Fix

Eski `/api/process` hattında MediaPipe `face_landmarker.task` dosyasını relative path ile açıyordu:

```python
model_asset_path="face_landmarker.task"
```

Windows’ta bu bazen bozuk path üretiyordu:

```text
site-packages/D:\...\face_landmarker.task
```

Bu yüzden `backend/face_detection.py` içinde model dosyası artık bytes olarak okunup MediaPipe’a verilir:

```python
model_buffer = model_path.read_bytes()

base_options = mp.tasks.BaseOptions(
    model_asset_buffer=model_buffer,
)
```

Bu eski sistemi bozmaz; sadece model yükleme path problemini çözer.

`face_landmarker.task` küçük bir dosyadır ve repo’ya commitlenmelidir.

Effect fonksiyon contract:

## Context Format

Face analysis sonrası context içinde beklenen ana alanlar:

```python
ctx = {
    "landmarks": ...,
    "landmarks_2d": ...,
    "landmarks_3d": ...,
    "parsing": ...,
    "masks": ...,
    "depth_map": ...,
    "pose": ...,
    "anchors": ...,
    "three_d": ...,
    "three_d_provider": ...
}
```

### Anchor format

```python
ctx["anchors"] = {
    "metrics": {
        "face_width": float,
        "face_height": float,
        "face_center": tuple,
    },

    "glasses": {
        "center": tuple,
        "eye_distance": float,
        "face_width": float,
        "width": float,
        "roll_deg": float,
        "left_eye": tuple,
        "right_eye": tuple,
        "nose_bridge": tuple,
    },

    "earrings": {
        "left": tuple,
        "right": tuple,
    },

    "necklace": {
        "center": tuple,
        "width": float,
        "chin": tuple,
    },
}
```

### 3D context format

DECA aktifse:

```python
ctx["three_d"] = {
    "provider": "deca_flame",
    "provider_priority": "deca_first",
    "is_true_3d": True,

    "vertices": np.ndarray,       # (5023, 3)
    "faces": np.ndarray,          # (9976, 3)
    "landmarks2d": np.ndarray,    # (68, 2)
    "landmarks3d": np.ndarray,    # (68, 3)

    "mesh": {
        "type": "deca_flame_mesh",
        "vertices": ...,
        "faces": ...,
        "vertex_count": 5023,
        "face_count": 9976,
        "is_true_3d": True,
    },

    "flame_params": dict,
    "camera": dict,
    "renderer_enabled": False,
}
```

DECA yoksa MediaPipe fallback kullanılmalıdır.
